# Error Codes

When using the Super73 electric bike, encountering error codes on the display screen can be frustrating. 

However, understanding what these error codes mean and how to resolve them can help ensure a smoother riding experience. 

Below are some common error codes that may appear on the Super73 display screen, along with solutions to fix them:

| Error Code | Issue | Solution |
| --- | --- | --- |
| E001 | Controller is dead | Replace Controller |
| E002 | ? |  |
| E003 | Torque Sensor / PAS Sensor Error | Replace Torque Sensor |
| E004 | ? |  |
| E005 | ? |  |
| E006 | Battery Undervoltage or Loose Cable near handlebar | Fully Charge Battery & Check Cables |
| E007 | Motor Fault | Check Motor Cable |
| E008 | Throttle Fault | Check Throttle Cable |
| E009 | Controller Fault | Check Controller Cables |
| E010 | ? |  |
| E011 | ? |  |
